# README

## 開発言語
* ruby 3.0.1
* Rails 6.0.6.1
## 就職Termの技術
* gem devise
* 1対1のメッセージ機能
## カリキュラム外の技術
* Natural Languadge
* gem rails_admin
* gem cancancan
* gem ransack
## 実行手順
```
$ git clone git@github.com:Uchiborii/CONKT.git
$ cd CONKT
$ bundle install
$ yarn install
$ rails db:create
$ rails db:migrate
$ rails s
```
